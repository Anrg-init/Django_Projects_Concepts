# Django Concepts & Implementations

This repository contains core Django concepts along with their **direct, practical implementations**.

It covers fundamental Django topics such as:
- MVT (Model–View–Template) architecture
- Forms and ModelForms
- Authentication and authorization
- Custom user model
- Function-based views
- Template inheritance
- Django ORM and database operations

The repository also includes **small projects focused on specific Django concepts**, demonstrating how each concept is used in real applications.

This project is intended for learning, practice, and reference while building Django-based applications.
